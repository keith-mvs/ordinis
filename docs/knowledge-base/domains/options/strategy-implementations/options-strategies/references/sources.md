## Usage Guidelines

### Source Priority Hierarchy

**For API Integration:**
1. Alpaca official documentation
2. Alpaca GitHub repositories
3. Community forum and Slack

**For Strategy Education:**
1. Alpaca Learn articles
2. OIC educational materials
3. CBOE strategy guides
4. Professional books (Hull, Natenberg)

**For Quantitative Modeling:**
1. Academic textbooks (Hull)
2. Python library documentation (NumPy, SciPy)
3. QuantLib documentation
4. Academic journals and papers

**For Compliance:**
1. SEC regulations and OCC rules
2. Alpaca Terms of Service
3. FINRA educational materials

### Citation Requirements

When documenting strategies or analysis:
- Always cite Alpaca Learn articles for platform-specific strategies
- Reference OCC documentation for margin and clearing rules
- Credit academic sources for pricing models and formulas
- Link to official API documentation for code examples

### Validation Process

1. **Primary Source**: Always use official Alpaca documentation first
2. **Cross-Reference**: Verify strategy concepts with OIC or CBOE
3. **Academic Validation**: Check formulas against Hull or academic papers
4. **Code Validation**: Compare implementations with Alpaca GitHub examples
