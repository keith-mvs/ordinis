## 8. Research Workflow

### 8.1 Validating a Trading Idea

```python
RESEARCH_CHECKLIST = [
    "1. Academic literature review",
    "   - Search Google Scholar, SSRN, NBER",
    "   - Find peer-reviewed papers on concept",
    "   - Note statistical significance and effect sizes",

    "2. Industry validation",
    "   - Check if practitioners use the concept",
    "   - Look for white papers from quant firms",
    "   - Review CFA Institute research",

    "3. Data requirements",
    "   - What data is needed?",
    "   - Is it point-in-time available?",
    "   - What is the data cost?",

    "4. Implementation feasibility",
    "   - Can it be coded as rules?",
    "   - What are the edge cases?",
    "   - How often does it trigger?",

    "5. Transaction cost analysis",
    "   - What is the expected turnover?",
    "   - What are spread/slippage costs?",
    "   - Is alpha > costs?",

    "6. Risk assessment",
    "   - What are the risks?",
    "   - When does it fail?",
    "   - What is the drawdown profile?"
]
```

## Key Takeaways

1. **Peer-reviewed first**: Validate ideas with academic research
2. **Credible sources only**: Use tiered reliability scoring
3. **Official data preferred**: SEC, Fed, exchanges are most reliable
4. **Cost matters**: Factor in data and transaction costs
5. **Point-in-time**: Avoid lookahead bias with proper data
6. **Stay current**: Markets evolve, research is ongoing
7. **Document everything**: Trace all logic to sources
