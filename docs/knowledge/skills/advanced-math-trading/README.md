# Advanced Math Trading Skillset

Use this index to pick the right skill; load only the skill you need.

- foundations-core — Probability, moments/tails, Bayes, statistical learning foundations.
- time-series-regimes — ARIMA/GARCH, stationarity, mean reversion, SDEs/discretization, regime detection.
- optimization-advanced — OCO, multi-objective (NSGA-II), DRO (CVaR/moment), MIP, Bayesian opt; convex/Markowitz/BL refs.
- portfolio-factors — Factor models, mean-variance, Black-Litterman, turnover/constraints.
- execution-control — Almgren–Chriss execution, control/MPC framing, queueing/order-book, exchange notes.
- signal-processing-features — Wavelets/EMD/Kalman/SSA, feature engineering, numerical methods.
- information-causal — Information theory (entropy/MI/TE), causal inference, Bayesian conditioning.
- robustness-risk — Tail/EVT, regularization, signal validation, pitfalls/best practices.
- simulation-inference — Monte Carlo, Brownian/jump processes, discretization schemes.
- microstructure-game — Game theory, network/systemic risk, queueing/order-book, exchange specifics.
